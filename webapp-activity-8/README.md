# 🧩 Web/App Codelab — Activity 8

## 측정 프로토콜을 사용하여 GA4에 웹 이벤트 전송

이번 Activity 8에서는 GA4 Measurement Protocol로 외부 이벤트를 전송하고, 파라미터 구성, 테스트 검증, 자동 호출까지 실습합니다.


## 📘 학습 목표

이번 코드랩에서 다루는 내용은 다음과 같습니다.

1. 일반
    - GA4 Measurement Protocol 개념 이해
    - MP 호출에 필요한 변수 수집 방법
    - 테스트 이벤트 전송 및 DebugView 검증
    - MP 호출 스크립트 작성 및 자동 전송 실습
2. 심화
    - 개인 프로젝트에 GA4 직접 적용
    - 버튼 클릭, 페이지 이동 등 이벤트 추적 구현
    - 이벤트 데이터를 분석하고 결과 정리
    - 블로그 또는 GitHub에 적용 사례 공유
